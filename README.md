---

## 1. Установка компилятора (GCC)
Для работы C на Windows нужен компилятор. Мы будем использовать **MinGW-w64**.

1.  **Скачайте установщик:** Перейдите на [GitHub проекта MSYS2](https://github.com) и скачайте `.exe` файл.
2.  **Установите:** Следуйте инструкциям установщика (обычно в папку `C:\msys64`).
3.  **Установите инструменты:** В открывшемся терминале MSYS2 введите команду:
    `pacman -S --needed base-devel mingw-w64-x86_64-toolchain`
    (Нажимайте `Enter`, чтобы подтвердить установку всех компонентов).

---

## 2. Добавление пути в PATH
Чтобы VS Code "увидел" компилятор, нужно прописать путь к нему в систему:

1.  Нажмите клавишу `Win` и введите в поиске: **"Изменение системных переменных среды"**.
2.  В появившемся окне нажмите кнопку **"Переменные среды..."** (внизу справа).
3.  В разделе **"Переменные пользователя"** (верхний блок) найдите строку `Path` и нажмите **"Изменить..."**.
4.  В новом окне нажмите кнопку **"Создать"** и вставьте путь к папке `bin` вашего компилятора. Обычно это:
    `C:\msys64\mingw64\bin`
5.  Нажмите **ОК** во всех окнах.
6.  **Перезагрузите VS Code**, чтобы изменения вступили в силу.

---

## 3. Удобный запуск из окна VS Code

Также для удобства установите расширения в VS Code (Ctrl+Shift+X): 
1. C/C++
2. C/C++ DevTools
3. C/C++ Extension Pack
4. C/C++ Themes
5. Code Runner

### Удобный запуск кода в VS Code
1. Нажмите `Ctrl + ,` (запятая), чтобы открыть настройки.
2. В поиске введите: `Run In Terminal`.
3. Поставьте галочку напротив пункта `Code-runner: Run In Terminal`.

### Как теперь запускать?
*   **Способ А:** Нажать на иконку Play в верхнем правом углу.
*   **Способ Б:** Нажать правую кнопку мыши в файле и выбрать **Run Code**.
*   **Способ В:** Нажать горячие клавиши `Ctrl + Alt + N`.
