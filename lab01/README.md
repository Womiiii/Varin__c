В этом файле инструкция, как посмотреть программы, которые я написл в рамках первой лабораторной работы. Номер задания = номер файла.

1. Выполните инструкции из общего README.md (он находится в папке "C_code", или на основной странице ветки "Extra branch" в GitHub)
2. Выберите интересующий вас файл из этой лабораторной
3. Запустите его с помощью кнопки "Run Code" (в верхнем правом углу) или сочетания кравиш Ctrl+Alt+N
4. Если что-то не получается, перепроверь ещё раз инструкции, если исправить не удалось, пиши в техподдержку.
