Ответы на вопросы:
• Как объявить структуру в языке C ? Вот так:

struct Student {
    char name[50];
    int age;
    float gpa;
};

• Что такое вложенные структуры и зачем они нужны 

Вложения - это некие переменные внутри структуры. В примере выше - это name, age, gpa.


• Как передать структуру в функцию? Вот так:

void functia (struct Student s)
 =


• Можно ли создавать массивы структур? Да, можно:

struct Student group[30];


• Как обратиться к элементу массива структур? Вот так:

group[0].age = 19; //обратились к 1 элементу масства, меняем возраст.


• Можно ли использовать enum внутри структуры? Да, можно:

struct student {
    enum Days {
        MONDAY = 1, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY, SUNDAY
    };
    enum Days day;
};


• Можно ли сравнивать структуры оператором == ?

Нет, нужно сравнивать только отдельные переменные в структурах.
